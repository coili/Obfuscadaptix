# Obfuscadaptix

Obfuscate your Adaptix beacon (only .exe for now) to bypass Windows Defender!

## Installation

```bash
git clone https://github.com/coili/Obfuscadaptix.git
```

```bash
cd Obfuscadaptix
```
```bash
pip install rich
```

## Usage 

```bash
python3 obfuscadaptix.py -f [path_to_beacon]
```
